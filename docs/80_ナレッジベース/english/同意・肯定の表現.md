# 同意・肯定の表現

"Yes" 以外にも文脈に応じた自然な表現がある。

## 提案への同意

| 表現 | ニュアンス |
|------|-----------|
| Sounds good | 「いいね、それで」。一番汎用的 |
| Sure | 「もちろん」。依頼への同意 |
| Makes sense | 「なるほど、理にかなっている」。論理的な同意 |

## 行動の許可

| 表現 | ニュアンス |
|------|-----------|
| Go ahead | 「どうぞ、やって」 |
| Please do | 「お願いします」。丁寧め |

## フォーマル度

カジュアル → フォーマルの順:

```
👍 / sg → Yeah → Sure → Sounds good → Yes, please → Certainly
```
