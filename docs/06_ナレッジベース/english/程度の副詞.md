# 程度の副詞

「どのくらい」を表す副詞。口語での使い分けが重要。

## pretty（結構、わりと）

"very" ほどではないが、それなりに。日本語の「結構」「わりと」に近い。

```
not → slightly → pretty → quite → very → extremely
                  ↑ ここ
```

エンジニア文脈での使用例:

| 表現 | ニュアンス |
|------|-----------|
| pretty straightforward | わりと素直 |
| pretty tricky | 結構ややこしい |
| pretty sure | ほぼ確信（確信度80%くらい） |
| pretty much done | だいたい終わった |
| pretty common pattern | よくあるパターン |

注意: カジュアルな表現。フォーマルな文書では "fairly" や "reasonably" を使う。
