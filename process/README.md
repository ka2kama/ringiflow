# process/

開発プロセスに関する知識を蓄積する。AI セッションに固有ではなく、プロジェクト全体の開発プロセスに関わる記録を格納する。

## サブディレクトリ

| ディレクトリ | 内容 |
|-------------|------|
| `improvements/` | 運用改善記録（問題の原因分析と対策） |
| `reports/` | 診断レポート（プロジェクト健全性の定点観測） |

## prompts/ との関係

AI セッション固有の知識（セッションログ、計画ファイル、操作レシピ）は [`prompts/`](../prompts/) に格納する。`process/` はプロジェクトの開発プロセス全般に関する知識を扱う。
