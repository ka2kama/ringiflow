# PR を Draft で作成しなかった

## 事象

Issue #208 の PR を作成する際、`gh pr create` を `--draft` オプションなしで実行した。

結果:
- PR が即座に Ready for Review 状態で作成された
- Claude Code Action の auto review が即座に実行された

## 原因分析

手順書（`docs/60_手順書/04_開発フロー/01_Issue駆動開発.md`）に `--draft` オプション付きのコマンド例が明記されているが、確認せずに PR を作成した。

## 対策

手順書に `--draft` の記載はあるが、現状それを参照せずに PR を作成している。

### 恒久対策

Issue #242 で PR 完了フローを明示的なチェックリストとして CLAUDE.md に追加する。手順書ではなく CLAUDE.md に記載することで、AI エージェントが常に参照する状態にする。
